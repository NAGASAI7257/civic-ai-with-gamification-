# 🚀 Civic AI with Gamification

An AI-powered civic issue reporting system enhanced with gamification to encourage citizen engagement, improve reporting efficiency, and provide real-time analytics.

---

## 📌 Overview

Civic AI with Gamification is a smart web platform that allows users to report civic problems such as potholes, garbage, and infrastructure issues. The system uses AI-based classification to automatically categorize and prioritize issues while motivating users through gamification elements like points, rankings, and achievements.

---

## ✨ Key Features

* 📝 **Smart Issue Reporting**
  Easily report civic problems with a simple interface

* 🧠 **AI-Based Classification (Rule-Based NLP)**
  Automatically detects issue category and priority

* 🎮 **Gamification System**

  * Points for reporting issues
  * Leaderboard system
  * User engagement tracking

* 📊 **Admin Dashboard**
  Monitor complaints, users, and system performance

* 📈 **Real-Time Analytics**
  Visual insights using charts and statistics

* 🌙 **Modern UI/UX**
  Dark mode, animations, and mobile-friendly design

---

## 🛠️ Tech Stack

## ▶️ Run Locally

1. Clone the repository
   git clone https://github.com/your-username/civic-ai-gamified.git

2. Navigate to project folder
   cd civic-ai-gamified

3. Install dependencies
   pip install flask

4. Run the application
   python app.py

5. Open in browser
   http://127.0.0.1:5000/

## 🧠 How AI Works

This system uses a rule-based Natural Language Processing approach:

* Keyword detection from user input
* Automatic category classification
* Priority assignment based on severity

✔ Lightweight
✔ No external APIs
